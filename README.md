# OSSFinalProject

## mini project 주제
  - 간단한 카페용 키오스크
<br/>

## mini project 에 대한 소개
  - 카페의 음료 메뉴를 관리하는 키오스크를 구현하였습니다.
  - 다양한 검색 기능을 제공합니다.
  - 각 메뉴의 총 주문을 관리합니다.

<br/>

## 추가될 기능 리스트

- 메뉴 조회
- 메뉴 추가 (오류 입력 방지 시스템 추가 / 자동 입력 시스템 추가)
- 메뉴 변경
- 메뉴 삭제
- 메뉴 검색 (검색 단어가 포함된)
- 가격대로 검색
- 종류별로 검색 (핫/아이스)
- 총 판매 가능 갯수 별로 검색
- 파일에 저장
